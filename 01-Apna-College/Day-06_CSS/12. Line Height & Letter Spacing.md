
# 📏 4. line-height
Sets the vertical space between lines of text.

## ✅Example:
```css
p {
  line-height: 1.5;  /* can be unitless, % or px */
}
```

# 🔠 5. letter-spacing
Adjusts spacing between characters (letters).

## ✅Example:
```css
h2 {
  letter-spacing: 2px;
}
```

# 🔤 6. font-size
Sets the size of the text.

## ✅Example:
```css
p {
  font-size: 16px;  /* can be px, em, rem, %, etc. */
}
```

The **Navbar** is a responsive navigation header built with Bootstrap.  
It supports branding, navigation links, forms, buttons, and is responsive by default.

---

## âœ… Basic Syntax:

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Brand</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Features</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Pricing</a>
        </li>
        <li class="nav-item">
          <a class="nav-link disabled" tabindex="-1" aria-disabled="true">Disabled</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

### ðŸ§© Key Classes:
- navbar â†’ Base class for the navigation bar.
- navbar-expand-lg â†’ Navbar expands on large screens and collapses on smaller ones.
- navbar-light bg-light â†’ Light background with dark text.
- navbar-dark bg-dark â†’ Dark background with light text.
- navbar-brand â†’ Logo or brand name.
- nav-link â†’ Individual navigation links.
- navbar-toggler â†’ Hamburger icon for mobile view.
- collapse navbar-collapse â†’ Wraps collapsible menu items.
